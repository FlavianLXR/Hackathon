# Hackathon Project
## Description

Projet réalisé lors du Hackathon 2024.
### Objectif : créer un jeu vidéo permettant d'apprendre la cybersécurité de manière ludique.

Le jeu s’adresse à différents profils de joueurs :

👶 Débutant – découverte des bases de la cybersécurité

🧑‍💻 Intermédiaire – compréhension des risques et bonnes pratiques

🧠 Expert – défis avancés liés à la sécurité informatique

L’idée est de gamifier l’apprentissage afin de sensibiliser aussi bien les gamers que les non-gamers aux enjeux de la protection des données.

## Objectifs du projet

Sensibiliser à la cybersécurité

Apprendre les bonnes pratiques de sécurité

Utiliser le jeu vidéo comme outil pédagogique

Proposer différents niveaux de difficulté

## Concepts abordés

Exemples de notions pouvant être apprises dans le jeu :
- Protection des données

- Phishing

- Gestion des mots de passe

- Attaques informatiques

- Bonnes pratiques de sécurité
